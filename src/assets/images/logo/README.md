# Logo Assets

## How to Add Your Company Logo

1. **Save your logo files in this folder:**
   - `logo.png` - Main logo (recommended: 200x200px or 512x512px, transparent background)
   - `logo-white.png` - White version for dark backgrounds (optional)
   - `logo-icon.png` - Icon only version (recommended: 100x100px)

2. **Recommended specifications:**
   - Format: PNG with transparent background
   - Size: 200x200px to 512x512px (square)
   - File size: Under 100KB for optimal performance
   - Color: Match your brand colors (purple/blue gradient recommended)

3. **Current usage:**
   - Header/Navigation bar
   - Footer
   - Favicon (browser tab icon)
   - Social media sharing

## File Naming Convention

- `logo.png` - Full logo with text
- `logo-icon.png` - Icon/symbol only
- `logo-white.png` - White version
- `logo-dark.png` - Dark version

## Path Reference

Use this path in your Angular components:
```
assets/images/logo/logo.png
```

Or in SCSS:
```scss
background-image: url('/assets/images/logo/logo.png');
```
